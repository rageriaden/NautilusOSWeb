About NautilusOS

NautilusOS is an innovative web-based operating system that brings the full desktop experience to your browser. Boot it up, log in, and interact with it just like a real desktop environment.

Everything runs client-side with no backend or external data storage. Built with pure HTML, CSS, and JavaScript, with no frameworks, and no dependencies aside from icons and fonts. It features three different proxy browsers, a complete desktop environment, multiple user accounts, an app store, and advanced cloaking and anti-monitoring features not found in other proxies.

Need a proxy? We've got three!

Each using different technologies to ensure you can always access the content you need

Nautilus Browser

The primary browser built into the OS with optimized performance and integrated features for seamless browsing experience.

Ultraviolet

Advanced proxy technology for bypassing restrictions with enhanced security and maximum compatibility.

Helios

An alternative, though less advanced, browser that uses CORS proxy technology for redundancy and accessibility when other methods are blocked.

Key Features

Everything you need for a complete desktop operating system experience

Desktop Environment

Enjoy a fully animated desktop experience with draggable and resizable windows, a taskbar, a searchable Start menu with app grid, a custom right-click context menu, and full window controls including minimize, maximize, and close.

Advanced Cloaking

Advanced cloaking features unmatched by other proxies, including tab blackout, anti-monitoring tools, automatic tab appearance rotation, customizable blackout delays, and manual control over tab titles and icons for maximum discretion.

Customization

A powerful theming system featuring numerous prebuilt themes, exportable and importable profiles for backup and sharing, adjustable window transparency, animation controls, and deep personalization.

AI Assistant

Built-in AI assistant integrated directly into the OS. Capable of answering questions, helping with tasks, and even controlling system functions with your explicit approval for automation.

Multi-User

Create and manage multiple user accounts with different editable permissions. Choose between superuser with full access or standard users with restricted capabilities. Each user gets their own desktop, settings, and app preferences.

App Ecosystem

Includes preinstalled apps and a built-in app store to discover more. Included in the store are dev-built games, apps, and themes, as well as a full suite of community-made apps proted directly from the community repo. Install new apps and extend Nautilus's functionality easily.

Notifications

A built-in notification system keeps you informed about system events, app updates, and important messages, while the quick action menu provides instant access to common tasks and settings.

File Explorer

Nautilus features a fully-fledged file explorer for managing your virtual files and folders. Create, delete, move, and organize files with an intuitive interface similar to traditional desktop OS file managers.

Media Tools

Includes a built-in audio player, screenshot capture tool, and image viewer, delivering a complete media suite without leaving Nautilus.

Built-in Games

Collection of preinstalled games for entertainment when you need a break. Play directly within the OS without external dependencies or downloads.

Profile Management

Export and import your complete Nautilus profile including settings, themes, apps, and preferences. Backup your setup or share configurations across devices easily.

Privacy Focused

All of your data is stored locally in your browser, on your device. No external servers, no tracking, no analytics. Your privacy is completely protected with a client-side only operation.
